## Learning Log
A record of what I'm learning as I work through a Cognitive Science degree (AI concentration) at Northwestern coming from a CS background, starting mostly from scratch on the cog sci side.

Not a portfolio. Just notes, questions, and things I've taken away.

## Structure
Each quarter gets a section. Each week gets a few entries, one per topic I explored. For each topic I write what I learned, what surprised me, and where I'm still confused, all in one blurb. I'll add entries as I go.
